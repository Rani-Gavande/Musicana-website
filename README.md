# Musicana

Musicana is a Django-based music streaming web application designed to provide an enjoyable music listening experience. Users can browse artists, explore albums, discover trending music, create accounts, manage profiles, and listen to their favorite songs through a simple and responsive interface.

## Features

* User Registration and Login
* Music Streaming
* Artist and Album Browsing
* Trending Music Section
* User Profile Management
* Responsive Web Interface

## Technologies Used

* Python
* Django
* HTML
* CSS
* JavaScript
* SQLite

## Purpose

The project was developed to demonstrate full-stack web development skills using Django while creating a functional music streaming platform.
